Automated data cleaning and reporting workflow using Python.

- Missing value handling
- Duplicate removal
- Data standardization
- Automated Excel report generation
- Visual analytics

## Technologies Used
- Python
- Pandas
- Matplotlib
- OpenPyXL

## Workflow
Raw Data → Cleaning → Analysis → Reporting

## Outputs
- Cleaned CSV
- Excel Report
- Charts
